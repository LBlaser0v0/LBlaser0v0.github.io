Python学习记录————基础版③

一、函数的定义与操作

1.函数的定义

#一种将具有独立功能的代码块组织成一个整体，使其具有特殊功能的代码。
#使用函数可以加强代码的复用性，提高程序编写的效率。
#函数格式要严格控制缩进。
#函数的定义格式： 函数名最好见名知义。
def 函数名(形参1，形参2，形参3......):
     函数体

2.函数的调用

#函数只有被调用的时候才会被执行，调用几次执行几次。
#调用方式：直接 函数名(实参1，实参2，实参3......)。
#例如：
#编写一个打招呼的函数并调用它。
def say hello():
    print ("hello")
    print("bingbing")
    print("18岁")
#调用函数前，必须保证函数已经存在

3.函数的返回值（使用return）

#作用：
#函数可以返回一个值，这个值可以作为函数的输出，也可以不返回任何值。
#return会给执行者函数的返回值。
#函数的返回值可以作为表达式的一部分，也可以作为函数的输出。
#函数的返回值可以是任意类型的数据，包括字符串、数字、列表、元组、字典等。
#函数中遇到return后，函数就结束了，后面的代码不会再执行。

#1.返回单个值
def buy():
return "一桶水果茶"  #比C语言能返回的值的类型更多
buy()
print(buy())        #输出：一桶水果茶，相当于将函数的值设为返回值。

#2.返回多个值
def buy():
    return "一桶水果茶", 20
    return 20       #不会被执行。
print(buy())        #输出：('一桶水果茶', 20) 
#返回多个值时以元组的形式返回，且return后面的代码不再执行。

返回值的三种情况总结：

    #1.一个返回值也没有，返回的结果是None。
    #2.一个返回值，就把值返回给调用者。
    #3.多个返回值，以元组的形式返回给调用。

4.函数的参数

#定义格式：
# def 函数名(形参a，形参b) :
    #函数体
    #...(如a=1,b = 2)
#调用格式：
#函数名(实参1，实参2)       实参要与形参一一对应。

1.形参与实参

#只在函数体内有效，函数调用结束后就消失。
#初始未赋值，可以在函数体内具体赋值使用，或者在函数调用时用实参来传参赋值。
#形参与实参的关系与C语言很类似

#调用例子：
def add(a.b):       #a，b就是形参
    return a+b
    print(add(2.5)) #2，5就是实参
#传参a=2.b=5

2.必备参数（位置参数）

#含义：传递和定义参数的顺序及个数必须一致
#格式：def func (a,b)
def func(name.age.sex):
    print("姓名:"name)
    print (age)
    print (sex)
func(18,'bingbing''女')#写了几个就必须要传几个，不可以多传，也不可以少传，并且顺序是一一对应的。
#若少些参数，则会报错。

3.默认参数（缺省参数）

#含义：定义函数时，给形参指定一个默认值，当调用函数时，如果没有传入实参，则使用默认值。 
#格式：deffunc(a=12):
def funb(a=8):
    print (a)
funb()       #输出：8
funb(200)    #输出：200
#注意：默认参数的值是静态的，不能修改。
#注意：所有的位置参数必须出现在默认参数前，包括函数定义和调用。
如：def func(a=1,b,c=3):  #错误，位置参数b必须出现在默认参数c前。
应该是：def func(b,a=1,c=3):  #正确。
